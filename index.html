Block level and inline elements

A Block element always starts on a new line and takes up the full width of a page.

    Examples of block level tags

  • Heading tags < h1 > to < h6 >
  • list (ordered, unordered description and list item) tags < ol > , < ul >, < dl >, < li >
  • pre-formatted text tags < pre >
  • blockquote tag < blockquote >

An inline elements does not cause a line break and does not take up the full width of a page.

    Examples of inline elements

  • anchor < a > tag
  • emphasis < em > tag
  • image < img > tag
